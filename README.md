# 🎯 Student Performance Predictor (Machine Learning)

This project predicts student exam scores based on daily study hours using Machine Learning.

---

## 📌 Overview

Can we estimate how well a student will perform just by knowing how many hours they study?

In this project, I built a predictive model using Linear Regression to explore the relationship between study time and exam performance.

---

## ⚙️ Tech Stack

- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Scikit-learn  

---

## 🧠 Approach

- Loaded and explored the dataset  
- Selected **study_hours_per_day** as input feature  
- Used **Linear Regression** for prediction  
- Split data into training and testing sets  
- Evaluated model using:
  - Mean Absolute Error (MAE)
  - Mean Squared Error (MSE)
  - Root Mean Squared Error (RMSE)
  - R² Score  

---

## 📊 Visualizations

- 📈 Histogram of exam score distribution  
- 🔵 Scatter plot of study hours vs exam score  
- 🔴 Regression line showing predicted trend  

---

## 📈 Results

- The model shows a strong linear relationship between study hours and exam score  
- Achieved reliable predictions with low error  
- Demonstrates how simple ML models can provide meaningful insights
