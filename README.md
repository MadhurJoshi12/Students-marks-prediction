# Students marks prediction
This project builds a machine learning model to predict student exam scores based on daily study hours using Scikit-learn. It demonstrates a complete workflow including data loading, model training, evaluation, and visualization.  A Linear Regression model is used to understand the relationship between study hours and exam performance. The model is evaluated using metrics such as Mean Absolute Error (MAE), Root Mean Squared Error (RMSE), and R² score to measure prediction accuracy.  Visualizations created with Matplotlib help in analyzing score distribution and the relationship between study hours and exam results.  This project reflects a practical application of supervised learning and forms a strong foundation for building more advanced machine learning systems.
